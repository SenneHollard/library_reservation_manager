# Library Reservation Manager — README for Dummies

This guide is for users who just want to run the app on their own laptop
with as little setup as possible.

---

## 1. Install Python

You need **Python 3.10 or higher**.

Download Python here:
https://www.python.org/downloads/

During installation:
- ✅ Check **"Add Python to PATH"**
- Click **Install**

Verify installation:
```bash
python --version
