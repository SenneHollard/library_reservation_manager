# Library Reservation Manager — README for Dummies

This guide is for users who just want to run the app on their own laptop
with as little setup as possible.

---

## 1. Install Python

You need **Python 3.10 or higher**.

Download Python here:
https://www.python.org/downloads/

During installation:
- ✅ Check **"Add Python to PATH"**
- Click **Install**

Verify installation:
```bash
python --version
```

### 2. Clone the repository
Open a terminal (Command Prompt / PowerShell / Terminal).
Go to the folder where you want the project:
```bash
cd Desktop
```

Clone the repository:
```bash
git clone https://github.com/SenneHollard/library_reservation_manager.git
```

Enter the project folder:
```bash
cd library_reservation_manager
```

If you don’t have git installed yet:
https://git-scm.com/downloads
